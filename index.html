0. 통합인증시스템 개요

대성산업 내 다양한 시스템을 접근하여 사용할 수 있는 내부 시스템입니다.

통합인증계정이 잠겼을 때, 해당 시스템의 비밀번호 찾기 기능을 통해 초기화가 가능합니다.(NAC도 동일합니다.)

[비즈모아포탈 바로가기] [통합인증시스템 바로가기]

1. 비즈모아포탈

기기 등록

크로미움 기반의 브라우저 환경에서 사용할 수 있는 신규 시스템입니다.

별도의 모바일 어플의 2차 인증을 통해 접근 가능합니다.

먼저 아래의 링크를 통해서 모바일 어플을 다운로드합니다.

[android 다운로드 페이지] [ios 다운로드 페이지]

이제 비즈모아포탈 상에서 로그인을 진행합니다.

입사자의 경우 ID, PW 모두 사번을 입력하고, 기존 임직원분들은 nleader 계정정보로 로그인합니다.

모바일 어플로 QR코드를 스캔하여 2차 인증을 완료합니다.

이후 로그인을 마저 진행하여 내부 시스템을 이용할 수 있습니다.

2. 구 통합인증페이지

Internet Explorer 기반의 브라우저 환경에서 사용할 수 있는 구 통합인증 시스템입니다.

법무커뮤니티, 해강연수원 등의 active x 를 사용하는 페이지 접근 시 사용합니다.

인터넷 옵션 설정 변경

먼저 인터넷 옵션 설정을 진행합니다.

Edge 설정변경

Microsoft Edge 설정을 변경하여 explorer 환경을 구성합니다.

해당 설정을 추가하여 Explore mode로 페이지에 접근이 가능합니다.

이후 Edge 설정가이드 를 통하여 통합인증프로그램 설치를 진행합니다.

설치 후 마저 로그인을 진행하여 시스템을 이용할 수 있습니다.
